# CloudVault Landing Page

A high-converting landing page for **CloudVault** — encrypted cloud storage built for small businesses in accounting, legal, and healthcare.

Built with [Next.js](https://nextjs.org/) 14, React 18, and pure CSS.

## Features

- Responsive design with mobile-first breakpoints
- Full landing page sections: Hero, Features, How It Works, Security & Compliance, Testimonials, Comparison Table, Pricing, FAQ, and CTA
- HIPAA compliance messaging
- Static export ready (for GitHub Pages, Vercel, Netlify, etc.)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Build & Export

```bash
# Build and export as static site
npm run build
```

The static output will be in the `out/` folder.

## Deploy

This project is configured for static export and can be deployed to:

- **Vercel** — connect your GitHub repo and deploy automatically
- **GitHub Pages** — push the `out/` folder contents
- **Netlify** — drag and drop the `out/` folder

## Tech Stack

- [Next.js 14](https://nextjs.org/)
- [React 18](https://react.dev/)
- Google Fonts (Poppins + Inter)
- Pure CSS (no Tailwind or UI libraries)
